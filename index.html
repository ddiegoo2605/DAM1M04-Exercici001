Benvinguts a CSS básic

Aquest és un text molt important perquè ens ajuda a entendre com funcionen els estils bàsics de CSS. Gràcies a aquest exemple podem comprovar com aplicar diferents regles d’estil a una mateixa pàgina web i veure com canvien els elements de manera immediata.

Aquest paràgraf combina diferents estils per mostrar text més gran, en negreta i subratllat. L’objectiu és practicar l’ús de múltiples classes al mateix element i comprovar com aquestes es poden acumular per aconseguir un resultat final més personalitzat.

  • Primera idea destacada: els estils CSS permeten donar color i forma als elements HTML sense modificar el seu contingut.
  • Segona idea interessant: podem reutilitzar classes i identificadors en diferents pàgines per mantenir un estil coherent.
  • Tercera idea final: una bona organització dels estils facilita el manteniment i l’actualització del projecte.
